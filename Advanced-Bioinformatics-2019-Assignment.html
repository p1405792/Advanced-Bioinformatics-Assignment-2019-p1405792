sumfun <- function(n) {
  answer <- sum(seq(by=1, from=5, to=n))
return(answer)}
sumfun(10)
## [1] 45
sumfun(20)
## [1] 200
sumfun(100)
## [1] 5040

To create the sumfun function I made n the variable being used with the result of the function under the moniker answer sumfun would then run the sum of a sequence of intergers (by=1) starting at 5 (from=5) until the sequence reached the variable n (to=n) To get the answers I used the sumfun function where n=10, then n=20, then n=100
